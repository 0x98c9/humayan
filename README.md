# Humayan Al Rosid – Portfolio

A modern, privacy-first portfolio and project showcase for Humayan Al Rosid, a 21-year-old BCA student from India. This site highlights offline-first tools, AI SaaS, and productivity utilities built for students and solopreneurs.

## 🌟 Features

- **Showcase of Projects:** AI tools, PDF utilities, productivity apps, and more
- **Modern UI:** Built with shadcn-ui, Tailwind CSS, and custom animations
- **Offline-First Philosophy:** Many tools work without a backend
- **Tech Stack Overview:** See the technologies and frameworks used
- **Contact & Collaboration:** Easy ways to connect for feedback or open source work

## 🛠️ Tech Stack

- **Frontend:** React, Vite, TypeScript, Tailwind CSS, shadcn-ui
- **Desktop:** Tauri, Rust
- **AI/Backend:** OpenAI, Supabase, Whisper, Node.js
- **Storage:** IndexedDB, LocalStorage, Supabase
- **Tools:** PDF.js, MathJax, Framer Motion
